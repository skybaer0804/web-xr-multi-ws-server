# Web XR 멀티플레이용 WebSocket 서버

Web XR 멀티플레이 환경을 위한 간단한 WebSocket 서버입니다.

## 설치

```bash
npm install
```

## 실행

```bash
npm start
```

서버는 기본적으로 `ws://localhost:8080`에서 실행됩니다.

포트를 변경하려면 환경 변수를 사용하세요:

```bash
PORT=3000 npm start
```

## 사용 방법

1. 서버를 실행합니다.
2. Web XR 클라이언트에서 `ws://localhost:8080`으로 연결합니다.
3. 클라이언트가 상태를 전송하면 모든 연결된 클라이언트에게 브로드캐스트됩니다.

## 기능

- 클라이언트 연결 관리
- 상태 동기화 (새 클라이언트에게 현재 상태 전송)
- 모든 클라이언트에게 브로드캐스트
- 에러 처리
